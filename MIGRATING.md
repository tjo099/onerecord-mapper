# Migrating between versions

This file documents one entry per minor release.

## v0.0 → v0.1.0 (initial install)

First public version. To install:

```bash
bun add github:tjo099/onerecord-mapper#v0.1.0
```

For production consumers, pin to a SHA after first integration:

```bash
bun add github:tjo099/onerecord-mapper#<commit-sha>
```

See `README.md` for usage. No prior version to migrate from.
